# My Profile Card

- This a simple newbie HTML & CSS project to help any beginner put his newly acquired skills in appropriate use practically

- The project acts like a digital shareable business card with a person's image, a brief description of them, what they do and links to their digital presence.

---

## Technical Stack
1. **Markup**-[HTML](https://developer.mozilla.org/en-US/docs/Web/HTML)
2. **Styling**-[CSS](https://developer.mozilla.org/en-US/docs/Web/CSS)
3. **Deployment & Hosting**-[GITHUB PAGES](https://docs.github.com/en/pages)

## QuickStart & Installation Guide
### Prerequisites 
- You only need a modern web browser and a code editor of your choice (e.g WebStorm, Notepad ++, Sublime Text, VS Code)

### Local Development 
1. **Clone the repository**
```bash
git clone [https://github.com/i-am-issaayman/My-Profile-Card.git](https://github.com/i-am-issaayman/My-Profile-Card.git)

cd My-Profile-Card
```
2.**Launch the Project**
* Open `index.html` directly in your browser, or
* Use the live Server Extentsion in VS Code/Cursor for hot-reloading.

### Project Structure
```
My-Profile-Card
├── index.html # Markup
├── styles.css # Styles
├── README.md # Main Project Markdown 
├── DESIGN.md # Design tokens 
├── pfp.jpg # Profile Picture
├── bg.jpg  # Background Image
└── favicon.ico # Favicon Graphic
```
